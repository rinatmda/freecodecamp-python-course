# Statistical Calculator

This project is a solution to the **Mean-Variance-Standard Deviation Calculator** project from FreeCodeCamp's 
[Data Analysis with Python](https://www.freecodecamp.org/learn/data-analysis-with-python/data-analysis-with-python-projects/mean-variance-standard-deviation-calculator) course.

It calculates statistical measures (mean, variance, standard deviation, min, max, sum) 
for a 3x3 matrix using Python and NumPy.

## Files

- `main.py` — main file to run the calculator
- `mean_var_std.py` — contains the `calculate()` function
- `test_module.py` — unit tests for the project
- `requirements.txt` — dependencies for the project
